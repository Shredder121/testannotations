= testannotations

:version: 0.1.0.BUILD-SNAPSHOT

Built-in setup/teardown rules for the masses

== Getting started

[source,java]
----
public class TestClass {

    @Rule
    @ClassRule
    public static final TestRule testRule = TestAnnotations.RULES;

}
----
